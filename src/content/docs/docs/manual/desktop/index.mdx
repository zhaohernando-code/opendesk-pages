# OpenDesk Desktop

OpenDesk Desktop 是一款由 OpenHarmony 社区孵化的开源智能桌面助手，为用户提供直观的图形界面来与 AI 助手交互。不同于普通的聊天机器人，它能**真正执行任务**——能读写文件、运行命令、管理日程、发送邮件，完成实际的工作。

用自然语言描述你的需求，它会自主规划执行路径，每一步都清晰可见。文件删除、代码修改等重要操作会主动询问确认，让你安心使用。支持 Windows、macOS 和鸿蒙 PC 等多平台。

---

## 它能做什么

| 领域 | 能力 |
|------|------|
| **文件与文档** | 读写、编辑、搜索（按名称/内容/语义），批量处理、格式转换 |
| **代码开发** | 理解项目结构、生成/修改代码、诊断修复 Bug、运行构建和测试 |
| **信息获取** | 联网搜索、学术文献检索、抓取网页内容 |
| **办公协作** | 管理日程和待办、收发邮件、整理笔记、自动化浏览器操作 |
| **智能规划** | 复杂任务自动拆分步骤，多任务并行推进，执行过程透明可控 |

---

## 适合谁用

| 身份 | 怎么用 |
|------|--------|
| **日常办公** | 写报告、做 PPT、分析数据、管邮件和日程、批量整理文件 |
| **软件开发** | 写代码、审查重构、理解项目架构、跑测试排 Bug |
| **深度研究** | 调研课题、检索论文、总结资料、生成分析报告 |

## 核心特点

| 特点 | 说明 |
|------|------|
| 🌟 **开源免费** | 背靠 OpenHarmony 社区，完全免费，无需订阅 |
| 🔒 **隐私安全** | 文件在本地处理，删除和覆盖等操作需确认后才执行 |
| ⚡ **自主执行** | 不只是对话——它真的会读写文件、运行命令、完成任务 |
| 🛠️ **能力丰富** | 文件、代码、邮件、日程、浏览器、知识库……覆盖工作各方面 |
| 🧠 **越用越懂** | 能记住你的偏好和习惯，跨会话保持上下文 |
| 🤝 **透明可控** | 执行过程实时可见，重要操作主动询问 |
