# 快速开始

5 分钟上手 OpenDesk，完成你的第一个任务。

---

## 一、启动 OpenDesk

双击 OpenDesk 图标，进入主界面。

| 区域 | 功能 |
|------|------|
| **侧边栏** | 任务列表，支持搜索，底部为设置入口 |
| **对话区** | 核心交互区：任务标题栏 + 消息列表 + 输入框 |
| **结果区** | 右侧面板：展示生成的文件、执行步骤、文件变更；可展开/收起，也可导出为文件 |


---

## 二、创建你的第一个任务

OpenDesk 理解自然语言——用一句话描述需求即可。

![](/assets/images/manual/quickstart_task.gif)

### 输入任务描述

示例：

- 「查看我今天的日程安排」
- 「整理我今天的邮件，并总结内容」
- 「分析这个项目的代码结构，帮我理解它的架构」

### 选择工作目录（可选）

1. 点击输入框右下角的「选择文件夹」按钮
2. 选择当前任务要使用的目录
3. OpenDesk 会基于该目录读取和处理文件

### 补充上下文（可选）

| 方式 | 说明 |
|------|------|
| `@` 引用 | 引用文件、文档、规则等上下文信息 |
| 粘贴截图 | `Ctrl/Cmd + V` 直接粘贴剪贴板中的图片 |
| 上传文件 | 拖拽文件到输入框 |
| 补充说明 | 在任务描述中明确目标、范围、约束和预期输出 |

建议补充：
- **目标**：要生成什么内容，或处理什么文件
- **输入**：需要分析的数据、参考的文档
- **输出格式**：希望得到什么格式的结果
- **约束**：风格要求、字数限制等

---

## 三、查看任务执行过程

任务创建后：

1. 新任务出现在左侧任务列表
2. 对话区持续展示执行过程和输出内容
3. 结果区展示生成的文件、执行步骤等
4. 你可以继续补充消息，或同时创建其他任务并行推进

对话区的展示内容包括：

- ✅ OpenDesk 的回复内容
- 🔄 执行阶段说明（如"正在分析数据"、"正在生成报告"）
- 📊 已生成的结果摘要
- 📝 可展开的中间步骤

---

## 四、查看任务结果

右侧结果区包含三类视图：

### 产物视图

本次任务生成的主要文件：文档（Markdown / Word）、数据文件（CSV / Excel）、图片（图表 / 截图）等。

### 变更视图

文件的具体变更内容：新增 / 删除 / 修改的高亮对比（diff）。

### 记忆视图
在当前任务执行过程中自动维护的结构化记忆文件，可用于后续上下文压缩。

---

## 五、继续对话与追问

任务完成后可以继续追问：

- 「再加一个饼图展示占比」
- 「把字体调大一点」
- 「导出为 PDF 格式」
- 「再加一个执行摘要部分」

追问的优势：无需重复背景，OpenDesk 保留已有上下文，基于已有内容快速调整。

---

## 六、多任务并行

点击「新建任务」即可创建多个任务，它们会并行执行。在侧边栏中可以搜索任务、按文件夹分组管理、查看状态和时间。

---

## 七、快速体验示例

| 类别 | 任务描述 |
|------|---------|
| 文件管理 | 「帮我在当前目录创建一个 README.md 文件，内容是项目介绍」 |
| 数据分析 | 「分析这个 CSV 文件，告诉我它有多少行数据，包含哪些字段」 |
| 代码理解 | 「阅读这个项目的 README 文件，总结项目的主要功能」 |
| 信息检索 | 「搜索关于 AI Agent 的最新文章」 |
