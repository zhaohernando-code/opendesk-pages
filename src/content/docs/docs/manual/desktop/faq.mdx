# 常见问题

---

## 基础问题

| 问题 | 回答 |
|------|------|
| **OpenDesk 是什么？** | OpenHarmony 社区孵化的开源 AI 桌面助手——不只是聊天，它能真正帮你执行任务 |
| **收费吗？** | 完全免费开源，无需订阅 |
| **支持哪些系统？** | HarmonyOS、macOS、Windows |
| **需要什么条件？** | 支持的设备 + 网络连接（用于调用 AI 模型） |

## 安全与隐私

| 问题 | 回答 |
|------|------|
| **数据会上传云端吗？** | 不会。文件操作都在本地完成，敏感数据不会外传 |
| **它会自动删我的文件吗？** | 不会。删除、覆盖等风险操作都必须你明确确认后才执行 |
| **隐私有保障吗？** | 代码完全开源可审查；所有敏感操作需确认且有日志可追溯 |
| **如何更好地保护隐私？** | ① 避免让它处理含密码或密钥的文件 ② 重要文件先备份 ③ 定期查看执行日志 |

## 任务执行

| 问题 | 回答 |
|------|------|
| **它怎么理解我说的话？** | 自然语言理解 → 自动分析目标 → 拆解步骤 → 逐步执行 |
| **能做哪些事？** | 文件管理、代码开发、数据分析、邮件处理、日程管理、知识检索、浏览器自动化等 |
| **任务失败了怎么办？** | 自动分析原因并尝试修复；无法解决时会主动告诉你 |
| **一般要多久？** | 简单任务几秒，复杂任务几分钟，大多数在 1-5 分钟内完成 |
| **能同时做多个任务吗？** | 可以，同时创建多个任务并行推进 |

## 文件操作

| 问题 | 回答 |
|------|------|
| **支持哪些文件格式？** | 文本类：代码（.js/.ts/.py 等）、文档（.md/.txt/.doc/.docx）、数据（.csv/.json/.xml）、配置（.env/.ini）等 |
| **能处理图片和 PDF 吗？** | 主要处理文本文件；图片、视频、PDF 等可通过专用功能或脚本处理 |
| **怎么指定工作目录？** | 创建任务时点击「选择文件夹」即可；不指定则使用默认目录 |
| **它会动我目录外的文件吗？** | 不会，只操作你指定的目录范围内的文件 |

## 代码开发

| 问题 | 回答 |
|------|------|
| **能写代码吗？** | 能：生成代码、补全片段、重构模块、修复 Bug、加注释和文档 |
| **代码质量怎么样？** | 遵循最佳实践：完整可运行、含必要注释、符合编码规范、包含安全检查 |
| **能看懂我的项目吗？** | 能：分析项目结构、理解架构设计、查找定义、分析依赖关系、生成文档 |
| **能跑命令吗？** | 支持，可以运行构建、测试、安装依赖等各种命令 |

## 邮件与日程

| 问题 | 回答 |
|------|------|
| **怎么管理邮件？** | 通过邮件技能：查看/搜索/回复/标记/收藏/删除邮件，支持多种邮箱协议 |
| **会自动发邮件吗？** | 可以帮你起草回复内容，发送前会让你确认并可以修改 |
| **怎么管理日程？** | 通过日程技能：查看/添加/修改/删除日程和会议 |
| **可以设置提醒吗？** | 可以设置定时提醒或周期性重复提醒 |

## 技能与扩展

| 问题 | 回答 |
|------|------|
| **有哪些技能？** | 邮箱、日程、笔记、浏览器、知识库、设备、人设、搜索、编程教学、全局待办、消息渠道、MCP 服务等 17+ 个，详见[技能系统](/docs/manual/desktop/skills) |
| **怎么用技能？** | 大多数情况直接说需求即可，OpenDesk 自动识别调用对应技能 |
| **怎么添加新技能？** | 通过技能管理功能：安装新技能 → 配置参数 → 使用 |
| **能接微信 / 飞书吗？** | 可以，通过消息渠道技能支持微信、飞书、WhatsApp 等多平台接入 |

## 性能问题

| 问题 | 回答 |
|------|------|
| **执行太慢怎么办？** | 常见原因：网络延迟、文件过大、任务复杂。建议：检查网络、限定文件范围、拆分大任务 |
| **怎么用得更高效？** | 说清目标 + 提供上下文 + 并行多任务 + 了解能力边界 |
| **占用系统资源多吗？** | 不高。主要消耗在 AI 调用和文件操作上 |

## 排查问题

| 问题 | 可能原因 | 怎么解决 |
|------|---------|---------|
| **读不了文件** | 路径不对 / 文件不存在 / 权限不够 | 检查路径拼写、确认工作目录、检查权限设置 |
| **命令执行失败** | 命令写错 / 权限不足 / 缺少环境配置 | 核对命令名称、检查权限、补充环境配置 |
| **打不开网页** | 没网 / 网站限制了访问 / 地址错了 | 检查网络连接、核对地址、尝试其他网站 |
| **回答不准确** | 给的上下文不够 / 需求太模糊 | 补充背景信息、说清楚具体要求、重试一次 |

## 和其他产品对比

### vs ChatGPT

| 维度 | OpenDesk | ChatGPT |
|------|----------|---------|
| 能力范围 | 能实际执行任务（操作文件、运行命令） | 只能对话 |
| 开源与否 | 完全开源 | 闭源商业产品 |

### vs Cursor / Devin

| 维度 | OpenDesk | Cursor | Devin |
|------|----------|--------|-------|
| 开源 | ✅ 完全开源 | ❌ 闭源 | ❌ 闭源 |
| 适用场景 | 办公+开发等多场景 | 以代码开发为主 | 以代码开发为主 |
| 扩展方式 | 技能系统（动态扩展） | 插件受限 | 功能固定 |
