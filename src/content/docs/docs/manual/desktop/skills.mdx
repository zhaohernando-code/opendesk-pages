# 技能系统

除了核心能力，OpenDesk 还通过**技能（Skills）**系统提供专业领域的扩展功能。每个技能专注一类场景——邮件、日程、笔记、浏览器等，按需自动调用。

![](/assets/images/manual/skill.gif)

---

## 技能总览

| 类别 | 技能 | 能做什么 | 典型场景 |
|------|------|---------|---------|
| **个性化** | 人设管理 | 定制 AI 的性格和沟通风格 | 让它更正式/更轻松/更有趣 |
| **邮箱** | 邮箱配置 | 添加和管理邮箱账号 | 接入工作邮箱、个人邮箱 |
| | 邮件操作 | 收发、搜索、回复、标记邮件 | 查邮件、写回信、搜归档 |
| **日程** | 日程管理 | 查看/添加/修改日程和会议 | 安排会议、设提醒 |
| **笔记** | 笔本管理 | 创建、编辑、搜索、整理笔记 | 知识管理、会议记录 |
| **浏览器** | 浏览器操作 | 打开网页、点击交互、提取数据 | 自动填表、采集信息、截图 |
| | 浏览器维护 | 清理历史记录和数据 | 隐私保护 |
| **知识库** | 知识库配置 | 接入企业知识库 | 连接公司内部文档系统 |
| | 知识库检索 | 搜索知识库内容并回答问题 | 查制度、找规范、检索文档 |
| **搜索** | 搜索配置 | 自定义搜索引擎和搜索源 | 切换搜索引擎、添加专业搜索 |
| **开发** | 编程教学 | 基于代码库的互动式编程学习 | 学新语言、深入理解框架 |
| **效率** | 全局待办 | 设置定时或周期性的提醒事项 | 每日站会提醒、周期性任务 |
| **消息** | 消息渠道 | 接入微信、飞书等通讯平台 | 多端消息同步、移动办公 |
| **设备** | 设备管理 | 管理智能设备节点 | IoT 设备控制、智能家居 |
| **扩展** | 技能管理 | 安装、更新、移除技能 | 按需扩展能力 |
| | MCP 服务 | 接入外部工具和服务 | 第三方集成、自定义工具链 |

## 怎么用

### 最简单：直接说需求

大多数情况下，你只需要自然描述需求，OpenDesk 会自动识别并调用对应技能：

- 「查看我今天的邮件」→ 自动使用邮件技能
- 「帮我加一个明天下午三点的会议」→ 自动使用日程技能
- 「搜索我的笔记里关于 AI 的内容」→ 自动使用笔记技能

### 遇到歧义时：指定技能名

如果需求可能被理解为多种操作，可以明确指定：

- 「用邮件技能帮我看一下未读邮件」
- 「用浏览器打开这个页面，把表格数据提取出来」

## 技能组合

复杂任务往往需要多个技能配合：

| 任务 | 用到的技能 | 流程 |
|------|-----------|------|
| 安排会议并通知 | 日程 + 邮件 + 笔记 | 创建日程 → 发通知邮件 → 做会议记录模板 |
| 整理一周工作总结 | 笔记 + 文件处理 + 邮件 | 搜集上周笔记 → 整合为报告 → 发送 |
| 从网上采集数据存档 | 浏览器 + 文件处理 + 知识库 | 网页提取 → 本地保存 → 入知识库 |

---

## 扩展更多技能

OpenDesk 的技能可以持续扩展：

- 通过技能管理安装社区贡献的新技能
- 支持接入 MCP 标准的外部服务和工具
- 企业用户可开发定制化内部技能
